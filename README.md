# ✈️ Flight Schedule App

A traveller-first web application built using Skyscanner's **Backpack Design System**.

## 🚀 Features
- Interactive flight date selection using BpkCalendar
- Clean UI following Skyscanner's brand guidelines
- Responsive layout with Backpack components

## 🛠️ Built With
- React (Create React App 3.4.1)
- Backpack Design System by Skyscanner
- bpk-component-calendar
- bpk-component-button
- SCSS for custom styling

## 📦 Installation
npm install bpk-component-calendar --save-dev

## ▶️ Run the App
npm start

Then open http://localhost:3000

## 📁 Key Files
- src/App.js — Main application with calendar integration
- src/App.scss — Custom styling with Skyscanner brand colours

## 🎨 Design System
Built with [Backpack](https://backpack.github.io/) — 
Skyscanner's open-source design system for 
building traveller-first experiences.
